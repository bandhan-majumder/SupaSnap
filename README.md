## Get started

1. Install dependencies

   ```bash
   yarn install
   ```
2. Setup .env.local

   ```bash
   cp .env.example .env.local
   ```

3. Setup supabase locally and setup the credentials (reference: see [docs](https://supabase.com/docs/guides/local-development/cli/getting-started))

4. Start the app

   ```bash
   npx expo start

   # or

   yarn android
   ```
